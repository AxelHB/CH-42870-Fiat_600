SI NODE, LES CREA UNA CARPETA DE NODE, CREAMOS EN LA RAIZ UN ".git_ignore" Y HACEMOS LO SIGUIENTE:

1-cortamos la carpeta de node y el package de json y lo movemos a otro lado fuera de la raiz.

2-Hacemos un "git add, commit y push" para enviar todo al repo sin esos archivos.

3-Antes de agregar los archivos que sacamos fuera, dentro del ".git_ignore" creamos las lineas con lo que querramos ignorar: /node_modules   /package.json

4-Ahora sí agregamos nuevamente los archivos y hacemos un "git add, commit y push" para enviar todo con esos archivos ignorados.
